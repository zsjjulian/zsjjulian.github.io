静态成员

不为某个对象独享，而是为类的所有对象共享的数据，属于类，不属于任何一个对象

  • 关键字 static
  • 形式：static 数据类型 静态数据成员名
  • 只能类内声明，类外初始化。
  • 静态成员函数可以直接访问类中说明的静态成员，但是不能直接访问类中的非静态成员函数，若要访问，需要参数传递的方式获取对象，由对象访问
  • 静态成员没有this指针

    用静态成员记录学生成绩和人数,静态成员是共享的，所以每次创建对象，人数都会增加1

友元

不是类的成员，但是访问类的所有成员

  • 关键字 friend
  • 声明的外界对象既可以是另一个类的成员函数，也可以是不属于任何类的一般函数，称为友元函数
  • 友元也可以是整个的一个类，称为友元类
  • 友元关系没有传递性，而且是单向的。1.B是A的友元，C是B的友元，但是C不是A的友元，2.B可以访问A的私有成员，但是A不可以访问B的私有成员

常对象与常成员

常对象

c++对常对象使用极为苛刻，不允许常对象调用任何类的成员函数，并且常对象定义后，在生命周期内不允许改变

  • const 类型 对象名
  • 类型 const 对象名
  • const对象不能被赋值，必须在定义时由构造函数初始化
  • const对象不能访问非常成员函数，只能访问常成员函数

常数据成员

常数据成员和常成员函数

  • const 数据类型 数据成员名
  • 数据类型 const 数据成员名
  • 同样必须初始化，且只能通过构造函数的初始化列表进行，并且不能被更新。
