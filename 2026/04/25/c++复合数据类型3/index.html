-union
-（1）所有成员共用一块空间，叫共用体；
-（2）用最大的成员的为公共空间；
-（3）所有成员低地址端对齐；
-（4）用的小端存储；
-（5）修改其中空间的值可能会改变其他成员的值，毕竟共用

小端存储是指 低数据位-低地址位；高数据位-高地址位

大端存储是指 低数据位-高地址位；高数据位-低地址位

#include <iostream>
union p
{
	unsigned char c[6];//一般把最大的放最前面
	int a;
	short d;
};
int main()
{
	p number{{0x12,0x34,0x56,0x78,0x9a,0xbc}};//只能初始化一个成员，默认是第一个
	std::cout<<std::hex<<number.a<<' '<<number.d<<std::endl;
	
    //赋值
	number.c[3]=0xff;
	std::cout<<std::hex<<number.a<<' '<<number.d<<std::endl;
	return 0;
}
//运行结果：
//78563412 3412
//ff563412 3412
