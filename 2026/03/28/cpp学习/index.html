C++的基础入门

名字空间

对于名字空间的使用大致有4种方式 :

请一定要酌情使用上述四种方式，单个项目均可用上述四种，且1方法比较省力；但是写多个项目时尽量不要将命名空间写入头部，会引起名字空间污染，本质上也就是重名了，编译器会报错

也可以自己命名名字空间
比如 ：

#include <iostream>
namespace cge//对于变量a的使用与上面四种相同
{
    int a=12;
}
using namespace cge;
int main()//可以命名多个名字空间，但内部不要同时命名相同的变量，在main函数打开时会发生冲突并报错
{
    using namespace std;
    cout<<a<<endl;
}
