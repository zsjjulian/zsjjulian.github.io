类

class

类也是一种数据结构，用class这种类型定义的变量叫对象,类型组成丰富，class也是struct功能的延伸class里面有访问权限，它们的唯一区别：class需要手写public，struct天生就是public。

函数定义和声明、构造函数、初始化列表

访问控制：

  • private: //私有的，类内成员才能访问使用
  • protected: //受保护的
  • public: //公有的
  • 顺序没有限制，数量没有限制，通过关键字保持一体。

    构造函数：创建对象时系统自动调用，函数名与类名保持一致。

对于引用变量，const变量这类必须初始化的成员，必须要用初始化列表

指针对象需要申请空间才能自动调用构造函数

  • Test* p;
  • p=new Test;

析构函数(~Test)

当对象的生命周期结束后自动调用，析构函数负责对象内部资源的清理，而delete负责释放由 new 分配的动态内存，并自动调用该对象的析构函数(内存释放)。

  • new：分配内存 + 调用构造函数。
  • delete：调用析构函数 + 释放内存。
  • 析构函数中可以释放对象成员指针指向的内存
  • 析构函数不能释放对象自身，否则会发生无限递归：析构函数释放对象本身，而对象本身生命周期结束时自动调用析构函数，陷入死循环。
  • 后创建的对象先析构，先创建的对象后析构，全局位置的对象最后析构

对象数组的申请与释放

delete[]释放对象数组，带[]才能为每个对象调用析构函数

this指针

c++结构体和类都有这个this指针，c语言没有这个指针

  • 装着当前对象的地址，在类内直接提供了地址，不需要&，只能在类内使用

拷贝函数

本质也是构造函数，与其他构造函数构成重载关系，只有参数不同，都无返回值

介绍

发生拷贝的情况：

  • 一个对象给另一个新对象初始化时
  • 函数传参时（按值传递时）
  • 函数按值返回对象（理论会，但c++17标准后直接优化了）（若用一个对象去接住函数的返回值，也可能不会发生拷贝构造）

应用

浅拷贝和深拷贝

浅拷贝就是平时一般用的，系统默认的，如果是深拷贝需要手动写。一般成员变量使用浅拷贝即可，又指针成员才需要深拷贝，因为如果需要复制指针成员，需要另外开辟一块新空间，避免两个指针指向同一块空间
